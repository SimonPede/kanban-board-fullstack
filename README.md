# Fullstack Kanban Board

Ein interaktives Kanban-Board, entwickelt im Rahmen des Studiums **Medieninformatik**. Dieses Projekt demonstriert das Zusammenspiel zwischen einem modernen Frontend (Vue.js) und einem REST-API-Backend (Node.js/Express).

## Features
- **Spalten-Ansicht:** Visualisierung von Aufgaben in verschiedenen Status (To Do, In Progress, Done).
- **Task Management:** Erstellen von neuen Aufgaben über ein modales Fenster.
- **Tag-System:** Kategorisierung von Aufgaben mit farbigen Badges.
- **Fullstack:** Eigener Express-Server zur Datenbereitstellung.

## Technologien
- **Frontend:** Vue.js 3 (Composition API), Bootstrap 5, Vite
- **Backend:** Node.js, Express.js
- **Tooling:** ESLint, Git/GitHub
